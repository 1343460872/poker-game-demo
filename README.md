### 游戏介绍
```
15个任意物品（可以是火柴牙签poker）
以下按牙签为例
 
将15根牙签
分成三行
每行自上而下（其实方向不限）分别是3、5、7根
 
安排两个玩家，每人可以在一轮内，在任意行拿任意根牙签，但不能跨行
 
拿最后一根牙签的人即为输家
```
